# The Weekly Rub Down — Weekly Run Sheet Prompt Template

## How to use this each week
Fill in the template below, attach your files, and send to Claude.
No need to type team codes — Claude reads the team logos directly from your screenshots.

---

## ATTACHMENTS NEEDED
1. Game By Game Matchup PDF (latest round)
2. Strength of Schedule PDF (latest round)
3. Waiver wire screenshots — one per position (FLB, CTW, 5-8, HFB, HOK, 2RF, FRF)
   → Tick the players you want included on each screenshot

---
